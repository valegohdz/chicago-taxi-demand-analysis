# Sprint 08 - Chicago Taxi Trips Analysis

## Description
Exploratory data analysis of taxi trips in Chicago to identify demand patterns and evaluate the impact of weather conditions.

## Project Structure
- notebooks/ : Jupyter notebook with analysis
- data/ : CSV files used in the project

## Technologies
- Python 3.11
- pandas
- numpy
- matplotlib
- seaborn
- scipy
- jupyter

## How to Run
1. Create and activate the environment:
   conda create -n sprint8 python=3.11
   conda activate sprint8

2. Install dependencies:
   pip install pandas numpy matplotlib seaborn scipy jupyter

3. Open the notebook and run all cells.
